# EcoQuery AI – Climate & Smart City Solution Engine

EcoQuery AI is an AI-powered assistant that generates structured solutions for climate and smart city challenges.

## Live Demo
https://climate-bot--joycodesall.replit.app

## Description
This application allows users to explore climate technology and smart city solutions through an AI-powered chat interface. It provides structured responses including executive summaries, solution tables, and implementation insights.

## Technologies Used
- OpenAI API
- React
- Node.js
- Replit
